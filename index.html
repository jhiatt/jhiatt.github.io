Points Kanban

Want to work on a project with a group of strangers? Points Kanban is a kanban system like Trello for coordinating work on a project with any number of independent developers. Points Kanban allocates points to cards so that the money or ownership resulting from the project can be allocated fairly among contributors. Points Kanban is coded with a Node.js/Express backend and a React/Redux front end on a Mongoose database.
